# Student Performance Analysis

##  Project Description
This project analyzes and predicts student performance based on study hours, attendance, and marks using machine learning techniques.

##  Features
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Model training
- Prediction of student results

##  Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

##  Dataset
The dataset includes the following features:
- Study Hours
- Attendance
- Previous Marks
- Internal Marks
- Final Result (Target)

##  How to Run

### Option 1: Run in Google Colab (Recommended)
1. Open the `.ipynb` file in Google Colab
2. Click on **Runtime → Run all**
3. The code will execute and show outputs (graphs + predictions)

### Option 2: Run on Local Machine
1. Install required libraries:
   pip install pandas numpy scikit-learn matplotlib

2. Open terminal in project folder

3. Run:
   python filename.ipynb

(Note: It is recommended to run this project in Google Colab or Jupyter Notebook)

##  Output
- Visualizations (graphs) from EDA
- Model training results
- Prediction of student performance (Pass/Fail or Grade)
